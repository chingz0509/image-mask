# 隐私截图打码器

一个在浏览器本地运行的截图隐私打码工具。适合在发送截图前，快速遮盖手机号、邮箱、身份证号、银行卡号、头像、地址等敏感信息。

## 功能

- 上传或拖拽图片
- 手动框选需要打码的区域
- 自动识别手机号、邮箱、身份证号、银行卡号和疑似地址
- 支持马赛克、黑条、模糊三种打码方式
- 支持删除单个识别框或删除当前图片
- 导出处理后的 PNG

## 隐私说明

图片处理在浏览器本地完成，不上传服务器。

自动识别功能使用浏览器端 OCR。首次使用自动识别时，需要加载 Tesseract.js CDN 资源。

## 使用方式

直接打开 `index.html`，或通过 GitHub Pages 访问发布后的网页。
